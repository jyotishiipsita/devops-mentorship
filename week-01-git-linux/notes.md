## Date started: 2026-03-14

## Git object model
- A blob stores file contents
- A tree stores directory listings
- A commit stores snapshot + parent + author
- A branch is just a 40-byte file pointing to a commit SHA

## Commands learned
- git cat-file -t / -p : inspect objects
- git rebase -i : interactive rebase
- git reflog : recover anything
- git cherry-pick : apply single commits
